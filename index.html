Primely Origin

Primely is an analysis tool in development by me, that aims to deliver my colleague and other employees some intelligence from the sequential study of their paychecks. This repo converts PDF paycheck files into JSON format and exports them in multiple approaches such as storing to MongoDB and visualizing by pandas package.

Primely Engine

The backend of Primely is deployed on AWS. Core Engine and API Engine are the two engines implemented on AWS Lambda. The Core converts original PDF file into JSON files which are uploaded from the client's device. Moreover, the API extract, transform and load the designated data group called by the client's device.
